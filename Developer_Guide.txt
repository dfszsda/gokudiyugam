Gokudiyugam App - Developer Documentation (v1.9.2)
================================================

1. Technical Stack:
   - Language: Kotlin
   - UI Framework: Jetpack Compose (Material 3)
   - Database/Auth: Firebase (Firestore, Auth, App Check, Cloud Messaging)
   - AI Integration:
     - Google Generative AI SDK (Gemini 1.5 Flash) for advanced content tasks.
     - Google ML Kit Translation (Offline & Online) for "Smart" language switching.
     - Custom "Lipi" mapping engine in AIService.kt for phonetic transliteration.
   - Media:
     - Media3 Exoplayer for background Kirtan playback.
     - YouTube Player library (Pierfrancesco Soffritti) for integrated video content.
   - External Sync: Google Sheets API via Apps Script and Google Drive API (v3) for dynamic media/document management.
   - Data Parsing: Gson for JSON serialization.

2. Architecture:
   - Pattern: MVVM (Model-View-ViewModel).
   - Core Package Structure:
     - `ui/screens/`: Compose UI definitions (e.g., HomeScreen, AdminPanel).
     - `model/`: Data classes (UserProfile, SabhaMeeting, SabhaTopic).
     - `ai/`: AIService for translation and content safety.
     - `security/`: Biometric and Permission logic.
     - `drive/`: Google Drive integration.

3. Key Components:
   - AIService.kt: Manages dual-mode translation (ML Kit Smart Translation + Manual Barakhadi Mapping).
   - PreferenceManager.kt: Handles SharedPreferences, user session caching, and granular screen permissions.
   - AnalyticsManager.kt: Logs screen views and events via Firebase Analytics.
   - BiometricManager.kt: Standardized interface for system-level biometric prompts.

4. Permission & Role System:
   - Roles: HOST, SUB_HOST, NORMAL.
   - Permissions: Controlled per screen (e.g., `screen_sabha_timetable`, `screen_admin_slider`).
   - HOST has master access; SUB_HOST access is configurable via the Admin Panel.

5. Content Management:
   - Dynamic Home Slider: Managed through AdminPanelScreen, stored in Firestore/Firebase Storage.
   - Sabha Timetable: Managed in SabhaDetailScreen with AI-assisted translation for topics.
   - Media: Centralized handling of Kirtan playlists and Video data via MediaViewModel.

6. Deployment & Configuration:
   - Target SDK: 36 (Android 15/16 preview ready).
   - Min SDK: 26 (Android 8.0).
   - Requires `google-services.json` and a valid Gemini API Key in `BuildConfig`.
   - Proguard enabled for release builds with specific exclusions for gRPC and Firebase.
