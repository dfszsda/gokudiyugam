Gokudiyugam Project - Technical Report Workflows

Detailed system documentation for project submission

1. System Architecture Overview

graph TD subgraph "Frontend (Mobile App)" UI[Jetpack Compose / M3] VM[View Models] Repo[Repository] end subgraph "Security & AI" Sec[Firebase App Check] AI[Gemini 1.5 Flash] end subgraph "Backend (Cloud)" Auth[Firebase Auth] DB[(Firestore)] ST[(Cloud Storage)] end UI --- VM --- Repo Repo --- Auth & DB & ST Repo --- AI DB --- Sec

Technical Specifications:

  • Frontend: Built with 100% Kotlin and Jetpack Compose for a modern, reactive UI. Uses MVVM architecture for clean separation of concerns.
  • AI Integration: Leverages Gemini 1.5 Flash for intelligent content processing and Google ML Kit for on-device language translation.
  • Backend: Fully serverless architecture using Firebase Ecosystem (Firestore for NoSQL data, Cloud Storage for media).
Kotlin Jetpack Compose Firebase Gemini AI
